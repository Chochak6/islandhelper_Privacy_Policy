# 岛助手隐私说明

隐私政策（Privacy Policy）

最后更新日期：2026-02-26

感谢你使用「岛助手」。我们非常重视你的隐私与个人信息保护。本政策将说明我们会收集哪些信息、如何使用、如何共享以及你如何联系我们。

⸻

1. 我们收集的信息

1.1 设备标识符（Device ID）

为实现账号识别、使用次数统计与订阅状态关联等功能，本应用会生成并使用设备标识符（例如设备标识/设备唯一标识符），并将该标识符上传至我们的服务器，用于提供应用核心功能与服务连续性。

用途：
	•	识别设备并维护应用功能状态
	•	统计每日使用次数/配额管理
	•	与订阅权益状态关联（功能实现层面）

1.2 扫描相关信息（文本与时间）

当你使用截图识别、拍照识别或手动输入功能时，本应用会记录：
	•	你扫描的文本内容
	•	你最后一次扫描/识别的时间

这些信息会用于：
	•	在灵动岛/实时活动中展示与更新内容
	•	维持你的卡片显示与编辑状态
	•	帮助你在使用过程中快速找回最近一次内容

说明：我们仅收集与应用功能相关的必要信息，不会要求你提供姓名、手机号、通讯录等与本应用功能无关的信息。

⸻

2. 我们如何使用信息

我们使用所收集的信息用于以下目的：
	•	提供文本识别、卡片展示与实时活动功能
	•	保存并更新你最近一次使用记录（如最近一次扫描文本与时间）
	•	进行必要的功能性统计（例如使用次数限制与配额刷新）
	•	提供订阅权益的验证与恢复（见第 3 节）

⸻

3. 订阅与第三方服务（RevenueCat）

本应用提供订阅服务。订阅购买与扣费由 Apple 通过你的 Apple ID 完成。我们不会获取或存储你的支付信息（例如银行卡信息）。

为管理订阅状态与权益验证，本应用使用 RevenueCat 作为订阅管理服务提供商。RevenueCat 可能处理与你的订阅相关的信息（例如订阅状态、购买验证所需信息），以便向本应用返回你的订阅权益结果。

⸻

4. 信息共享与披露

我们不会出售你的个人信息。仅在以下情况下共享必要信息：
	•	与 RevenueCat 共享：用于订阅管理与权益验证
	•	法律要求：当法律法规、司法机关或监管机构要求时，我们可能依法披露必要信息

⸻

5. 数据存储与保留
	•	我们会在实现本应用功能所需的期限内保存相关数据。
	•	你可以通过卸载应用或清除相关数据（如应用内提供删除/清除功能）来移除本地数据。
	•	与订阅相关的数据处理以 Apple 与 RevenueCat 的规则为准。

⸻

6. 你的权利

你可以随时：
	•	停止使用本应用的识别功能
	•	删除应用以移除本地数据
	•	在系统设置中管理或取消订阅（Apple ID → 订阅）

如你希望咨询或请求处理与数据相关的问题，可通过下方联系方式联系我们。

⸻

7. 儿童隐私

本应用不以儿童为目标用户。我们不会主动收集儿童的个人信息。如你认为我们可能无意收集了相关信息，请联系我们进行处理。

⸻

8. 本政策的更新

我们可能会不定期更新本隐私政策。更新后将通过本页面公布，并更新“最后更新日期”。
